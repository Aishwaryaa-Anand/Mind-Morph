Introduction

  • AI-powered web application for MBTI personality type prediction
  • Combines 3 analysis methods: Questionnaires, Text Analysis, Social Media
  • Uses advanced ML algorithms and Natural Language Processing
  • Achieves 80.36% prediction accuracy (vs 55-65% industry standard)
  • Real-time results with comprehensive personality insights
  • Features: History tracking, PDF reports, JWT authentication

Existing System and Drawbacks

Existing Systems

  • Keyword-based sentiment analysis
  • Single-method personality tests
  • Social media analysis tools
  • Manual MBTI questionnaires
  • Language-specific models

Drawbacks

  • Low accuracy (55-65%)
  • No context understanding
  • Cannot handle sarcasm/humor
  • Privacy concerns
  • Limited to single data source
  • Requires technical expertise
  • Self-reporting bias

Problem Statement

  • Existing systems lack multi-modal personality analysis
  • Single-source predictions are less reliable and prone to bias
  • Need for real-time, accurate personality insights
  • Gap in user-friendly AI-powered personality tools
  • Challenge: Combine NLP, ML, and psychology effectively
Our Goal: Develop a comprehensive system that analyzes personality through multiple channels with 80%+ accuracy using ensemble machine learning

Detailed Module Description

Feature Extraction Module

BERT embeddings (768 dim) + CountVectorizer (5,000 dim) + Linguistic features (20 dim) = 5,788 total features

Binary Classifiers

4 independent classifiers for I/E, N/S, T/F, J/P dimensions trained on 8,675 labeled samples

Ensemble Prediction

Soft voting across 3 algorithms (LogReg + XGBoost + RandomForest) with confidence scoring

Results Service

MBTI type mapping, personality insights generation, PDF export, history tracking

Machine Learning Pipeline

1
